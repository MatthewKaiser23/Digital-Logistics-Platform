What this project is about:

  • This project aims to create a comprehensive digital logistics platform designed to manage the global delivery of South African wine. The project not only addresses the logistical complexities of global wine distribution but also contributes to the enhancement of the wine industry's efficiency and customer satisfaction. This endeavour involves a sophisticated computational process that will result in the generation of a rich dataset. This dataset will enable us to achieve several key objectives.
  • The project's core functionality will be realized through a computational process that encompasses web scraping, the manipulation of airline and road traffic APIs, and data collection on weather conditions in the chosen cities. Additionally, big data computation will play a significant role, utilizing the power of the Hadoop and Spark ecosystems to compute optimal routes from airports to hotels and restaurants and generate insightful statistics pertaining to the wine delivery system.
